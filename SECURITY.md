# Security Policy

## Supported Versions
Currently, Kel's Algorithm is in active development. Only the latest release tag is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.8  | :white_check_mark: |
| < v0.1.8| :x:                |

## Reporting a Vulnerability
If you discover a mathematical flaw in the Lattice Engine, a collision vector in the Hash Chaining, or a bypass in the Quarantine Sandbox, do **NOT** open a public issue.

Please report all security vulnerabilities privately to the repository maintainer.
