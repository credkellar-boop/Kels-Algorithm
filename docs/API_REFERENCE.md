# API Reference

## SpiderOrchestrator
The main entry point for Kel's Algorithm.
- `initialize_system()`: Mounts the spider legs and sets the active OID.
- `initiate_cycle_shift()`: Forces the topology to jump to a new mathematical plane.

## NonLinearBinder
- `derive_bound_key(classical: bytes, quantum: bytes) -> bytes`: Returns a 512-bit unified key.
